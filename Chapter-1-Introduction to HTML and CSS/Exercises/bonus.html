HTML & CSS Quick-Reference Guide

A handy reference for commonly used HTML tags and CSS properties.

HTML Tags

Tag Description Example
<p> Defines a paragraph. 
<p>This is a paragraph.</p>
<a> Defines a hyperlink, which is used to link from one page to another. 
<a href="#">This is a link</a>
<img> Embeds an image in the HTML page. 
<img src="example.jpg" alt="Example Image">
<ul> <ol> <li> Defines lists (unordered or ordered) and list items. 
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
<h1> to <h6> Defines headings of different levels, from largest (h1) to smallest (h6). 
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<form>, <input>, <button> Defines form elements such as form, input fields, and buttons. 
<form>
    <input type="text" placeholder="Enter text">
    <button type="submit">Submit</button>
</form>
<div> Defines a division or section in an HTML document. 
<div>This is a division.</div>
<span> Defines a section in a document. 
<span>This is a span.</span>
<table>, <tr>, <th>, <td> Defines a table and its elements (table row, table header, table data). 
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>
<strong>, <em> Defines text with strong importance (bold) and emphasized text (italic). 
<strong>Important Text</strong>
<em>Emphasized Text</em>

CSS Properties

Property Description Example
font-family Sets the font family for text. font-family: Arial, sans-serif;
font-size Sets the font size. font-size: 16px;
color Sets the text color. color: #333;
background-color Sets the background color. background-color: #f9f9f9;
margin Sets the margin around an element. margin: 10px;
padding Sets the padding inside an element. padding: 10px;
border Sets the border of an element. border: 1px solid #ccc;
text-align Sets the alignment of text. text-align: center;
